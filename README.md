# Manager Box

## Sistema de gerenciamento de estoques para pequenos lojistas

### Sobre o projeto

A ideia do Manager Box surgiu a partir da observação de uma necessidade por parte de pequenos comerciantes e lojistas
de trazer o controle de seu estoque de mercadorias para o meio digital e automatizar etapas de rotina.

<br>

Para direcionar o desenvolvimento da plataforma, foi realizada uma pesquisa do tipo <i>survey</i>
com o público-alvo da ideia, analisando as opniões dos usuários de sistemas de gerenciamento de estoque
já existentes no mercado levando em conta os seguintes aspectos:

<ul>
    <li>Suporte e Funcionalidades</li>
    <li>Usabilidade</li>
    <li>Eficiência e Desempenho</li>
</ul>

Ao analisar os resultados, constatou-se que que os fatores de Suporte e Funcionalidades
foram os que atingiram o menor nível de satisfação, seguido por Usabilidade e Eficiência e Desempenho,
em ordem crescente.

Portanto, torna-se evidente a necessidade de um produto que satisfaça bem todos esses aspectos.
